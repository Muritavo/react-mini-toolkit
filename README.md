# react-mini-toolkit
A toolkit with scripts of stuff I'm tired of copy/pasting or implementing from start

# How to use
Install it using
```npm i -g @muritavo/react-mini-toolkit```

Execute the CLI with the command
```react-mini-toolkit help```
